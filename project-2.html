Project Overview

This project aims to develop a predictive model to classify individuals as diabetic or non-diabetic using the Pima Indian Diabetes dataset. It begins with data exploration to understand the dataset's structure, visualize the outcome distribution, and examine correlations. Missing values and outliers in key features like glucose, BMI, and insulin are handled through imputation and filtering, ensuring data integrity. Feature selection focuses on health-related variables such as pregnancies, glucose, and blood pressure. Several machine learning algorithms, including Logistic Regression, Decision Trees, and Random Forests, are evaluated after splitting the data into training and testing sets. The ultimate goal is to identify the most effective model for diabetes prediction.

This project uses various machine learning methods to predict diabetes. Logistic Regression serves as a baseline for binary classification, w hile Support Vector Machines (SVM) and K-Nearest Neighbors (KNN) explore non-linear patterns. Decision Trees and Random Forests handle feature interactions and provide ensemble learning capabilities. Advanced methods like Gradient Boosting and Neural Networks a re also evaluated to identify the most effective model.
